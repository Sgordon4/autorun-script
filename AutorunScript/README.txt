AutorunMega is a simple autorun script
for any game, simulating only the 'w'
keypress. The text below describes how
to use the exe.

>> Running autorunMega:

Simply double-click on 'autorunMega.exe'
A small command prompt should appear with.
To stop the autorun script close the prompt.

P.S. If you would like a version without the
prompt contact me. It would be harder to 
close though.



>> Strafe toggle:

Enabled by default.
You can toggle the autorun from stopping 
when a/d (strafe keys) are pressed.



>> Keybindings for autorunMega:

Begin autorun: '=' and '\'

Stop autorun --
	Strafe_toggle off(default): 'w', 'a', 's', 'd'
	Strafe_toggle on 		  : 'w', 's'
	
Strafe toggle: '-'
